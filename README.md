# TrustSphere

TrustSphere is a frontend Trust Organization Management System built with React, Vite, React Router, Bootstrap 5, and Tailwind CSS. It simulates a professional trust dashboard for managing donations, beneficiaries, events, volunteers, and organizational preferences using local storage instead of a backend.

## Features

- Landing page with branding and project overview
- Dashboard with summary metrics and donation visualization
- Full CRUD management for donations, beneficiaries, events, and volunteers
- Search and filter page for donations, beneficiaries, and events
- Settings page with theme switching and local data reset
- Local storage persistence so changes remain after refresh
- Responsive interface using Bootstrap layout and Tailwind-powered styling

## Pages

- Landing Page
- Dashboard
- Donation Management
- Beneficiary Management
- Event Management
- Volunteer Management
- Search & Filter
- Profile / Settings

## Tech Stack

- React
- Vite
- React Router DOM

- Tailwind CSS
